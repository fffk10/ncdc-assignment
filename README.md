## 技術スタック

・React v18

・Next.js v14.1.0(App Router)

・tailwindCSS v3.3.0

## 起動方法

```bash
1. npm i

2. npm run dev
```

[http://localhost:3001](http://localhost:3001)にアクセスでページを確認できます。

[BE 側プロジェクト](https://github.com/ncdcdev/recruit-frontend)も合わせて起動してください。

## フォルダ構成

```
.
└── /src/app
    ├── page.tsx # Homeページ
    ├── /assets # 画像フォルダ
    ├── /components # コンポーネントフォルダ
    ├── /lib # 汎用処理フォルダ
    ├── /styles # css modulesフォルダ
    └── /types # 型定義フォルダ
```
